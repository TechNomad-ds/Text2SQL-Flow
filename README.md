# Text2SQL-Flow  
This repository contains the code implementation of Text2SQL-Flow.

## Repository Structure  
- `data` stores sample data for Text-to-SQL tasks.  
- `operators` contains the operators.  
- `pipelines` includes the implementation of the pipeline.  

## Installation  
You can install it by following these steps:  

```shell  
git clone https://github.com/OpenDCAI/DataFlow  
cd DataFlow  
pip install -e .  
```  

Then, run the pipeline code from the `pipelines` directory.
